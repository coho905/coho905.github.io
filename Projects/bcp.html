About the Project

PLEASE NOTE: the site will take a minute or two to load at first due to its set-up as running only when requested. I promise it will load eventually if you are patient.
I originally had the idea for this project when I came across a tumor analysis dataset on Kaggle.com while browsing during the "High Tech Hacks" Hackathon. I was curious to see if I could use the data to predict if a patient had a malignant or benign growth. The model was build during the hackathon and won 2nd place on the Innovation Track, while the resulting website was something that I built after the fact.
The model is a multi-layered perceptron neural network and was build using the scikit-learn library. It is approximately 97% accurate, and does better at predicting malignant types (which is more important to catch early).
The website itself was built with simple HTML and CSS. It was hosted on the free service onrender.com and is currently up and running. The back-end uses Flask to manage the user inputs and transfer them to the model properly. When all put together, this project provides a quick interface for doctors (and patients with the appropriate information) to determine whether a growth is malignant or benign via mammogram measurements.
