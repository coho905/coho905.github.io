About the Project

Unfortunately, due to the complicated nature of passing files through flask, this is not hosted online like the breast cancer app . However, the code is posted on the right, and you can clone the repo to run it locally.
This was developed for the Commonwealth Governor's School culminating project. The project was to create a website that would allow the user to input a picture of a skin legion and have it be classified as either malignant or benign. The project was built using the Flask framework, and the data was pulled from Kaggle.
After a year's worth of research into convolutional neural networks, which are based off of the human visual cortex, the model was put into development. Currently the model is approximately 92% accurate, and it was developed using the fast.ai framework. Feel free to download the code or look into the research paper (to be posted when I find the file from my SCPS hardrive).
