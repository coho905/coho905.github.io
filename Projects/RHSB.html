About the Project

I designed and implemented this project as part of the Senior Capstone project within Riverbend High School's computer science department. This system improvement was requested by the school's administration due to the increasing amounts of theft and drug deals taking place. Thank you to the University of Mary Washington team that graciously allowed me to use their servers for free.
The system has 2 parts: a web application for students to sign in from, and a back-end module to handle the database. The web app was build using HTML, CSS, and JavaScript, while the backend was build using Flask and Python's mariadb library for SQL database management. After a student enters their username and password, then the Studentvue API will determine what block it is, then it will determine the correct bathroom for you to use and check the necessary qualifications (bathroom not over capacity), and finally it will print your ticket. If you have been in the bathroom for more than 15 minutes, you will be checked out and the admin will be alerted. The system keeps track of all bathroom logs for the week and then will properly discard the data.
